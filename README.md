# Utami Food - Website Resmi 🍽️

Selamat datang di repositori website resmi **Utami Food**! 
Website ini berfungsi sebagai *landing page* dan katalog digital untuk usaha kuliner Utami Food, spesialis Nasi Bakar dan Ceker Mbeledos.

## 🌟 Fitur Utama
* **Katalog Menu Interaktif:** Menampilkan menu andalan (Nasi Bakar berbagai varian), Pelengkap, dan Minuman beserta harganya.
* **Sistem Pre-Order (Pesan Sekarang):** Form terintegrasi untuk pelanggan melakukan pemesanan (pre-order) lengkap dengan pilihan jumlah porsi dan hari/jam pesanan.
* **Testimoni Pelanggan:** Menampilkan ulasan positif dari pelanggan setia.
* **Profil Koki:** Memperkenalkan Master Chef di balik lezatnya hidangan Utami Food (Utami & Monik).
* **Desain Responsif:** Tampilan website yang dioptimalkan untuk perangkat *mobile*, tablet, maupun *desktop*.

## 🛠️ Teknologi yang Digunakan
* HTML5 & CSS3
* Bootstrap 5 (Framework CSS)
* JavaScript
* Font Awesome & Bootstrap Icons
* Animate.css & OwlCarousel

## 🚀 Cara Menjalankan Secara Lokal
1. *Clone* repositori ini ke komputer Anda.
2. Buka folder proyek.
3. Klik ganda pada file `index.html` untuk membukanya langsung di *browser* Anda.
